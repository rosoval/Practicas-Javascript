Variables y constantes

Variables Condicionales Bucles Arrays Muestra

Variables

let nombre = "Ana";
let edad = 25; 
edad = 30; // reasigna permitida
console.log("Hola, soy " + nombre + " y tengo " + edad + " años"); // muestra: "Hola, soy Ana y tengo 30 años."

Constantes

const TIEMPO_ESPERA = 5000; // no se puede modificar
// TIEMPO_ESPERA = 3000 // reasignación no permitida
console.log(TIEMPO_ESPERA); // muestra: 5000
