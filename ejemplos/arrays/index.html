Listas (arrays)

Variables Condicionales Bucles Array Muestra

Definición de array

let frutas = [ "manzana", "naranja", "plátano", "fresas" ];

console.log(frutas[0]); // "manzana"
console.log(frutas[1]); // "naranja"
console.log(frutas[2]); // "plátano"
console.log(frutas[3]); // "fresas"

Modificación de un elemento

frutas[2] = "pomelo";
console.log(frutas[2]); // "pomelo"

Cantidad de elementos

console.log(frutas.length); // 4

Agregar un elemento

frutas.push("mango");
console.log(frutas[0]); // "manzana"
console.log(frutas[1]); // "naranja"
console.log(frutas[2]); // "pomelo"
console.log(frutas[3]); // "fresas"
console.log(frutas[4]); // "mango"
